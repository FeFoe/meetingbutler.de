# Description Editor Simplification + Admin Dashboard Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the 15-field EventDetail form in termin.html with a single description textarea, and add a password-protected admin dashboard showing per-user event counts and token usage.

**Architecture:** Feature 1 is a pure frontend change to `public/termin.html` — the backend already handles `description` correctly. Feature 2 adds a `tokensUsed` column to the Event table, threads token counts from the LLM response through to the DB, adds a stats API endpoint, and provides a static HTML dashboard page.

**Tech Stack:** NestJS + TypeScript, Prisma 5, PostgreSQL, plain HTML/CSS/JS (no framework), OpenAI SDK

## Task 5: Add `/api/admin/stats` endpoint

**Files:**
- Modify: `src/admin/admin.controller.ts`

Add a new `GET /api/admin/stats` endpoint. Password check via query param compared to `ADMIN_DASHBOARD_PASSWORD` env var. No new guard needed — inline check is sufficient for this internal tool.

- [ ] **Step 1: Add the stats endpoint to `admin.controller.ts`**

Full updated file `src/admin/admin.controller.ts`:

```typescript
import { Controller, Get, Query, UnauthorizedException, UseGuards } from '@nestjs/common';
import { ConfigService } from '@nestjs/config';
import { PrismaService } from '../common/prisma.service';
import { AdminApiKeyGuard } from './admin-api-key.guard';

const GPT_NANO_COST_PER_TOKEN = 0.000003; // EUR, gpt-5.4-nano approximation

@Controller('admin')
export class AdminController {
  constructor(
    private prisma: PrismaService,
    private config: ConfigService,
  ) {}

  @Get('health')
  health() {
    return {
      status: 'ok',
      service: 'meetingbutler',
      timestamp: new Date().toISOString(),
      version: '1.0.0',
    };
  }

  @Get('stats')
  async stats(@Query('password') password: string) {
    const expected = this.config.get<string>('ADMIN_DASHBOARD_PASSWORD');
    if (!expected || password !== expected) {
      throw new UnauthorizedException('Falsches Passwort');
    }

    const users = await this.prisma.user.findMany({
      where: { verified: true },
      include: { events: { select: { tokensUsed: true } } },
      orderBy: { createdAt: 'desc' },
    });

    const rows = users.map((u) => ({
      firstName: u.firstName,
      lastName: u.lastName,
      email: u.email,
      eventCount: u.events.length,
      tokensUsed: u.events.reduce((sum, e) => sum + e.tokensUsed, 0),
    }));

    const totalTokens = rows.reduce((sum, r) => sum + r.tokensUsed, 0);
    const totalEvents = rows.reduce((sum, r) => sum + r.eventCount, 0);

    return {
      summary: {
        totalUsers: users.length,
        totalEvents,
        totalTokens,
        estimatedCostEur: +(totalTokens * GPT_NANO_COST_PER_TOKEN).toFixed(4),
      },
      users: rows,
    };
  }

  @UseGuards(AdminApiKeyGuard)
  @Get('raw-emails')
  async rawEmails() {
    return this.prisma.rawEmail.findMany({
      orderBy: { receivedAt: 'desc' },
      take: 50,
    });
  }

  @UseGuards(AdminApiKeyGuard)
  @Get('queues')
  queues() {
    return {
      note: 'Queue dashboard available if @bull-board is configured',
      queues: ['email-ingest', 'llm-extract', 'event-match', 'ics-generate', 'email-send'],
    };
  }
}
```

Note: `ConfigService` is now injected — check that `AdminModule` has `ConfigModule` available. Since `ConfigModule.forRoot({ isGlobal: true })` is set in `AppModule`, it's available everywhere.

<script>
  let allUsers = [];
  let sortKey = 'eventCount';
  let sortAsc = false;

  async function login() {
    const pw = document.getElementById('pw-input').value;
    if (!pw) return;

    try {
      const res = await fetch(`/api/admin/stats?password=${encodeURIComponent(pw)}`);
      if (res.status === 401) {
        document.getElementById('error-msg').textContent = 'Falsches Passwort';
        return;
      }
      if (!res.ok) throw new Error('Serverfehler');
      const data = await res.json();
      renderDashboard(data);
      document.getElementById('login-card').style.display = 'none';
      document.getElementById('content').style.display = 'block';

      document.getElementById('error-msg').textContent = e.message;

  function renderDashboard(data) {
    const s = data.summary;
    document.getElementById('summary').innerHTML = `
      <div class="stat-card"><div class="label">Nutzer</div><div class="value">${s.totalUsers}</div></div>
      <div class="stat-card"><div class="label">Events</div><div class="value">${s.totalEvents}</div></div>

    `;
    allUsers = data.users;
    renderTable();
  }

  function renderTable() {
    const sorted = [...allUsers].sort((a, b) => {
      const va = sortKey === 'name' ? (a.firstName + a.lastName).toLowerCase() : a[sortKey];
      const vb = sortKey === 'name' ? (b.firstName + b.lastName).toLowerCase() : b[sortKey];
      return sortAsc ? (va > vb ? 1 : -1) : (va < vb ? 1 : -1);

    document.getElementById('tbody').innerHTML = sorted.map(u => `
      <tr>
        <td>
          <div class="name">${esc(u.firstName)} ${esc(u.lastName)}</div>
          <div class="email">${esc(u.email)}</div>
        </td>
        <td class="num">${u.eventCount}</td>
        <td class="num">${u.tokensUsed.toLocaleString('de-DE')}</td>
      </tr>
    `).join('');
  }

  function sortBy(key) {
    if (sortKey === key) { sortAsc = !sortAsc; } else { sortKey = key; sortAsc = false; }
    renderTable();
  }

    return String(s || '').replace(/&/g,'&amp;').replace(/</g,'&lt;').replace(/>/g,'&gt;');
  }
